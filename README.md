# KB2026
Автор: Вейгандт Вероника
Описание: Создать GUI-приложение «Movie Library» для хранения информации о фильмах с фильтрацией, JSON и Git.
Реализовано: фильтрацию: по жанру, по году выпуска; сохранениее и загрузка данных через JSON; Проверка корректности ввода; Использован Git.
Для запуска используйте команду python main.py
Примеры использования: можно вести учёт фильмов, которые уже посмотрели; использовать приложение как Wishlist; использовать как мини-каталог семейной медиатеки.
