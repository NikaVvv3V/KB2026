## Описание программы
Movie Library - это простое desktop-приложение для ведения личной коллекции фильмов. Программа позволяет добавлять, удалять и фильтровать фильмы, а также сохранять данные в JSON файл.

### Возможности программы:
* Добавление фильмов с указанием названия, жанра, года выпуска и рейтинга
* Просмотр всех фильмов в виде таблицы
* Фильтрация фильмов по жанру (частичное совпадение)
* Фильтрация фильмов по году выпуска (точное совпадение)
* Удаление фильмов из коллекции
* Автоматическое сохранение данных в JSON файл
* Загрузка данных из JSON файла при запуске
* Валидация вводимых данных

### Проверка корректности ввода:
* **Год выпуска:** должен быть целым числом от 1900 до 2026
* **Рейтинг:** должен быть числом от 0 до 10 (можно использовать дробные числа)
